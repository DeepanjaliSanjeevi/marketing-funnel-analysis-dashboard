# Marketing Funnel Analysis Dashboard

## 📊 Project Overview

This project analyzes user behavior across a marketing funnel: View → Cart → Purchase.

## 🎯 Objective

To identify conversion rates and understand where users drop off in the funnel.

## 📈 Key Insights

* Majority of users only view products
* Very low conversion rate (~1–2%)
* High drop-off before purchase
* Funnel requires optimization

## 🛠 Tools Used

* Microsoft Excel (Pivot Tables, Charts)

## 📂 Files Included

* Excel Dashboard
* word Report

## 🚀 Outcome

This project demonstrates practical marketing funnel analysis and business insight generation.
